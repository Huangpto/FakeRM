# FakeRM
This is a repository created for the EMNLP 2026 submission of FakeRM. We only provide a samll subset of the dataset. The remaining data will be made available to reviewers upon reasonable request during the review process. If the paper is accepted, we will publicly release the complete dataset and benchmark code. This has been illustrated in Appendix C.
